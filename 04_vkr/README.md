# VKR Workspace

Это главный рабочий раздел будущей ВКР.

## Подпапки

- `00_drafts` — старые и переходные черновики.
- `01_outline` — структура, матрицы, gap-analysis.
- `02_rpz` — мастер-файлы и рабочая пояснительная записка.
- `03_appendices` — приложения и графические материалы.
- `04_slides` — презентация и раздаточные материалы.

## Текущая логика работы

1. Структуру ВКР держать в `01_outline`.
2. Основной текст ВКР собирать в `02_rpz`.
3. Все приложения складывать в `03_appendices`.
4. Материалы защиты собирать в `04_slides`.

## Ближайший содержательный план

- выделить разделы НИРС, которые переходят в ВКР без сильной переработки;
- оформить техническое задание;
- спроектировать рабочее проектирование;
- собрать доказательное апробирование;
- довести графическую часть до формата защиты.
