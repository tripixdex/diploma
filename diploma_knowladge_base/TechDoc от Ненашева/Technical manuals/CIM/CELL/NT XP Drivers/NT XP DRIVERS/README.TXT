Softlok International Limited - Mar 97

Device driver for Windows NT

Driver installation:
	
    Use on NT version 3.5/3.51 or 4.0
    Copy driver to system driver directory:
	copy softlok.sys %systemroot%\system32\drivers

    Setup registry (regini.exe is provided on this disk.):
	regini softlok.ini

    Shutdown system

    Reboot

    The driver is now installed and ready to use
