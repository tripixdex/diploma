# Pre-Hardware Policy Layer

This directory defines the reviewer-facing operating policy for the current AGV Denford software-only MVP.

It answers four questions:
- what the project is allowed to claim now,
- how the current contour may be operated now,
- what must not be claimed,
- what must be true before the first honest hardware phase starts.
