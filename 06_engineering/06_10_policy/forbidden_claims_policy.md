# Forbidden Claims Policy

## Claims That Must Not Be Made Now
- The project is hardware-ready.
- The project is safety-validated on the real AGV.
- The project is deployment-grade.
- The project is production-ready.
- Webots integration is implemented and evidenced.
- PostgreSQL runtime is implemented and evidenced.
- Docker deployment is implemented and evidenced.
- Mosquitto deployment baseline is implemented and evidenced.
- Raspberry Pi runtime binding is implemented and validated.
- Orange Pi portability is proven.
- Real AGV modernization is already deployed.

## Claims That Require Direct Evidence Before They Can Be Used
- Board-specific runtime support.
- Real GPIO control.
- Real motor/sensor interface support.
- Real safety-behavior validation.
- Real deployment packaging.
- Real lab-tested board portability.

## Wording Rule
- If the statement is roadmap-only, say roadmap-only.
- If the statement is abstraction-only, say abstraction-only.
- If the statement is local/dev/demo evidence only, say local/dev/demo only.
- If there is no direct artifact, do not imply completion.
