## Роль НИРС в текущем проекте

Текущий НИРС — это содержательная база для ВКР. Из него напрямую переходят:

- анализ предметной области;
- анализ задачи;
- концепция;
- требования;
- обзор объекта автоматизации;
- структурная схема;
- базовые алгоритмы.

До ВКР не хватает:

- формального ТЗ;
- рабочего проектирования;
- апробирования;
- полного комплекта защитных материалов.
