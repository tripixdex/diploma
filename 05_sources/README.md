# Sources

## Подход к источникам

Здесь источники разделены логически, а не только исторически.

- `05_00_raw_knowledge_base` — сырой наследованный архив дипломника.
- `05_01_primary_denford` — сюда будут выноситься самые важные первичные материалы по стенду и AGV.
- `05_02_secondary_research` — статьи, обзоры, вспомогательные исследования.
- `05_03_related_theses` — смежные ВКР и КП.

## Правило работы

Если источник становится реально опорным для текста ВКР, на него нужно:

1. завести заметку в Obsidian;
2. выписать полезные тезисы;
3. отметить, в какой раздел ВКР он идет;
4. при необходимости переместить или продублировать ссылку в curated-раздел.
