# Чеклист По ВКР

## Административное

- [ ] Уточнена окончательная тема ВКР
- [ ] Подготовлено задание на ВКР
- [ ] Подготовлен титульный лист
- [ ] Собран календарный план
- [ ] Подготовлены материалы для нормоконтроля

## Текст ВКР

- [ ] Есть раздел предпроектного обследования
- [ ] Есть раздел концептуального проектирования
- [ ] Есть формальное ТЗ
- [ ] Есть структурное проектирование
- [ ] Есть рабочее проектирование
- [ ] Есть апробирование
- [ ] Есть заключение
- [ ] Есть список источников
- [ ] Есть приложения

## Графическая часть

- [ ] Есть не менее 15 слайдов или эквивалент
- [ ] Не менее 40% графики оформлено по стандартам
- [ ] Все рисунки есть в тексте РПЗ и на них есть ссылки
- [ ] Подготовлена защита

## Нормоконтроль

- [ ] Оформление соответствует методичке
- [ ] Поля, шрифт, интервалы проверены
- [ ] Нумерация сквозная
- [ ] Приложения оформлены корректно
- [ ] Заимствование в допустимом диапазоне
